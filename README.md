# MitoCine
Este proyecto es el resultado final del Curso de Android con Kotlin 2020. Se trata de una aplicación Android que permite obtener información sobre películas en cartelera y a estrenarse.

## Funcionalidades
<ul>
<li>Obtener películas en cartelera: La aplicación se conecta al servidor de base de datos Backendless y utiliza su Rest-API para obtener información actualizada sobre las películas en cartelera y por estrenar.
</li>
<li>Arquitectura MVVM: El proyecto sigue la arquitectura Modelo-Vista-VistaModelo (MVVM), lo que facilita la separación de responsabilidades y mejora la organización del código.
</li>
<li>Interfaz de usuario intuitiva: La interfaz de usuario de la aplicación ha sido diseñada de manera intuitiva, proporcionando una experiencia agradable para el usuario.
</li>
</ul>
